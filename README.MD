# Introduction
```
.
├── build              : 打包程序的文件夹
├── dist               : 同上
├── pngpic.iconset     : 图标文件夹
├── pic                : 用到的图片
├── config.py          : 一些配置信息
├── core.py            : 主要数据存储交互
├── frames.py          : 主要界面
├── gui.py             : GUI管理类
├── icon.icns          : 程序图标
├── icon.png           : 程序图标1024x1024原图
├── main.py            : 主程序, app类所在
├── main.spec          : pyinstaller打包主配置文件
├── noname.cpp         : wxformbuilder的文件
├── noname.h           : 同上
├── readinglist.fbp    : 同上
├── pic.py             : 文本代码化的图片
├── registry.py        : 注册界面的装饰器
├── util.py            : 一些通用函数
```
